# 🌍 Global Development Clustering

## 📌 Problem Statement
Analyze global development data and group countries based on socio-economic indicators.

## 📊 Dataset
Includes:
- GDP
- Life expectancy
- Income
- Health expenditure
- etc.

## ⚙️ Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Scaling
- K-Means Clustering

## 📈 Results
- Countries grouped into clusters based on development level
- Identified patterns between developed and developing nations
